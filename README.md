# 🎬 CineMatch Pro

AI-powered movie recommendation system built using Streamlit and NLP.

## 🚀 Features
- Content-based filtering
- Cosine similarity
- Modern cinematic UI
- Interactive recommendations

## 🛠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas

## ▶ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
